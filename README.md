# NBA Player Performance Prediction 🏀

An end-to-end data analytics pipeline and machine learning model built to forecast NBA player scoring outputs based on tracking data, physical metrics, and efficiency ratings.

## 🚀 Executive Summary
* **The Goal**: Predict a player's true scoring average (Points Per Game) using physical data and on-court metrics.
* **The Dataset**: Over 11,000 player season profiles cleaned and processed using Pandas.
* **The Result**: Achieved an **R-squared of 0.72** using a Scikit-Learn baseline pipeline.

## 📊 Key Insights Found During EDA
* **Opportunity Rules Everything**: Usage Percentage (`usg_pct`) maintains the single strongest correlation with scoring volume (**0.75**), vastly outpacing pure scoring efficiency.
* **The Pace Multiplier**: Holding all other traits equal, an increase in player weight slightly decreased scoring output, reflecting the faster pace and high-volume perimeter output generated by leaner guards and wings.

## 🛠️ Performance Metrics & Simulation
* **Model Type**: Linear Regression Baseline
* **R-squared**: 0.72 (Explains 72% of the variance in league-wide scoring)
* **Real-World Case Test**: Fed a hypothetical high-usage 7-foot rookie profile into the trained weights, resulting in a predicted simulation output of `X` PPG.
